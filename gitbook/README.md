# FluxAuth Documentation

Welcome to FluxAuth - an open-source behavioral biometric authentication platform.

## What is FluxAuth?

FluxAuth provides continuous authentication by analyzing how users type and interact with their devices. Instead of relying solely on passwords, it creates a unique "typing fingerprint" for each user.

## Key Features

- **Continuous Authentication**: Monitor users throughout their session
- **AI-Powered Analysis**: Gemini AI explains security decisions
- **Bot Detection**: Identify automated attacks and scripts
- **Privacy-First**: No PII stored, only behavioral patterns
- **Open Source**: Free to use, modify, and deploy
